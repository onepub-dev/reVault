VaultDirectory

VaultDirectory class final

A writable, password-protected metadata store for one reVault installation.

Open or create it through Vault to manage profile keys, contacts, form definitions, encrypted key backups, remembered lockbox paths, and local access-slot labels. Lockbox file contents are stored separately.
