Vault

Vault class final

Entry point for encrypted lockboxes, keys, local vault metadata, the session agent, and the platform secret store.

Call load once when the application starts, then use the returned object to create or open lockboxes, manage recipient and signing keys, and access the local services used to remember vaults and credentials. See the repository README for a complete example and guidance on handling secret values.
